# Community Energy Orchestrator Frontend

React + Vite web interface for the Community Energy Orchestrator. Connects to the FastAPI backend to run community workflows and visualize energy analysis results.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 — requires the API running on port 8000.

## Scripts

- `npm run dev` — Development server with hot reload
- `npm run build` — Production build
- `npm run preview` — Preview production build
- `npm run lint` — ESLint
